# InSync Ecommerce CRM

## Environments

### Production

branch : `main`

domain : `main.insync-crm.com`

### Development

branch : `develop`

domain : `develop.insync-crm.com`

## Install

Some basic Git commands are:

```
$ git clone https://github.com/abendsoft/insync-crm.git
$ cd insync-crm
$ yarn install
```

## Start development

```
$ yarn run dev
```

## Build for production

```
$ yarn run build
```

## Run build for production

```
$ yarn start
```
