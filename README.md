# Awal HR Management System Tenant Dashbaord

## Environments

```
1. Production

==> Branch Name: main

2. Development

==> Branch Name: develop

```

## Install

```sh

git clone https://github.com/awal-solution/awal-hrms-tenant-dashbaord.git
cd  awal-hrms-tenant-dashbaord.git
yarn
```

## Start development

```sh
yarn dev
```

## Build for production

```sh
yarn build
```
